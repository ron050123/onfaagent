API Documentation

Integrate EmbedChat Pro with your applications using our REST API

API Overview

Base URL

All API requests should be made to:

https://your-domain.com/api

Response Format

All responses are returned in JSON format:

{
  "success": true,
  "data": { ... },
  "message": "Operation completed successfully"
}

Authentication

API Key Authentication

Include your API key in the Authorization header:

Authorization: Bearer YOUR_API_KEY

You can find your API key in the dashboard under Settings → API Keys

Session Authentication

For web applications, you can use session-based authentication:

Cookie: session=YOUR_SESSION_TOKEN

Analytics Endpoints

GET /api/analytics

Get analytics data for a specific bot (requires botId parameter)

Query Parameters:
  • • botId - Bot ID to get analytics for (required)
  • • startDate - Start date for analytics (YYYY-MM-DD, optional)
  • • endDate - End date for analytics (YYYY-MM-DD, optional)

Code Examples

JavaScript/Node.js

const axios = require('axios');

const apiClient = axios.create({
  baseURL: 'https://your-domain.com/api',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

// Send a message to the bot
async function sendMessage(message, botId) {
  try {
    const response = await apiClient.post('/chat', {
      message: message,
      botId: botId,
      userId: 'user_123',
      sessionId: 'session_456'
    });
    
    return response.data;
  } catch (error) {
    console.error('Error sending message:', error.response.data);
    throw error;
  }
}

// Get bot analytics
async function getBotAnalytics(botId, startDate, endDate) {
  try {
    const response = await apiClient.get('/analytics', {
      params: {
        botId: botId,
        startDate: startDate,
        endDate: endDate
      }
    });
    
    return response.data;
  } catch (error) {
    console.error('Error fetching analytics:', error.response.data);
    throw error;
  }
}

Python

import requests
import json

class EmbedChatAPI:
    def __init__(self, base_url, api_key):
        self.base_url = base_url
        self.headers = {
            'Authorization': f'Bearer {api_key}',
            'Content-Type': 'application/json'
        }
    
    def send_message(self, message, bot_id, user_id=None, session_id=None):
        """Send a message to the bot"""
        url = f"{self.base_url}/api/chat"
        data = {
            'message': message,
            'botId': bot_id,
            'userId': user_id or 'user_123',
            'sessionId': session_id or 'session_456'
        }
        
        response = requests.post(url, headers=self.headers, json=data)
        response.raise_for_status()
        return response.json()
    
    def get_bots(self):
        """Get all bots for the authenticated user"""
        url = f"{self.base_url}/api/bots"
        response = requests.get(url, headers=self.headers)
        response.raise_for_status()
        return response.json()
    
    def create_bot(self, name, description, settings=None):
        """Create a new bot"""
        url = f"{self.base_url}/api/bots"
        data = {
            'name': name,
            'description': description,
            'settings': settings or {}
        }
        
        response = requests.post(url, headers=self.headers, json=data)
        response.raise_for_status()
        return response.json()

# Usage example
api = EmbedChatAPI('https://your-domain.com', 'YOUR_API_KEY')

# Send a message
response = api.send_message("What are your business hours?", "bot_123")
print(response['data']['response'])

# Get all bots
bots = api.get_bots()
print(f"Found {len(bots['data'])} bots")

Rate Limiting

API requests are rate limited to ensure fair usage:

  • Chat Endpoint: 100 requests per minute per user
  • Bot Management: 50 requests per minute per user
  • Analytics: 30 requests per minute per user

Rate limit headers are included in responses:

X-RateLimit-Limit: 100
X-RateLimit-Remaining: 95
X-RateLimit-Reset: 1640995200

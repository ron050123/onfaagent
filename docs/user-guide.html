Getting Your Embed Code

Once your bot is configured, get the embed code:

<script src="https://your-domain.com/bot.js" data-bot="YOUR_BOT_ID"></script>

Copy this code and paste it into your website's HTML before the closing </body> tag.

Important Notes

  • • Replace "YOUR_BOT_ID" with your actual bot ID from the dashboard
  • • Replace "your-domain.com" with your actual domain
  • • The widget will appear in the bottom-right corner of your website
  • • The script automatically handles responsive design and mobile compatibility

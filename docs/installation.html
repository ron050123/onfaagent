Dashboard Setup

Step 1: Extract the Files

After purchasing from CodeCanyon, you'll receive a main_files.zip file. Extract it to get the EmbedChat Pro files:

# Extract the main_files.zip
unzip main_files.zip

# This will give you EmbedChatPro.zip
unzip EmbedChatPro.zip

# Navigate to the extracted folder
cd EmbedChatPro

Note: Make sure you have all the files extracted properly. You should see folders like app, components, lib, and files like package.json.

Important: OpenAI API Key Required

You need to add your OpenAI API key in two places:

  • In the environment file (Step 3 below)
  • In the dashboard settings (after login)

Step 2: Install Dependencies

Install all required dependencies including the new multi-source knowledge base libraries:

npm install

Step 3: Configure Environment Variables

Copy the environment template and configure your settings:

cp env.example .env.local

Edit the .env.local file with your configuration:

# Database Configuration
MONGODB_URI="mongodb://localhost:27017/chatbotdb"
# Or for MongoDB Atlas (cloud):
# MONGODB_URI="mongodb+srv://username:password@cluster.mongodb.net/chatbotdb"

# OpenAI API Configuration
OPENAI_API_KEY="your_openai_key_here"

# NextAuth Configuration
NEXTAUTH_SECRET="your_random_secret_here"
NEXTAUTH_URL="http://localhost:3000"
# For production, change to your domain:
# NEXTAUTH_URL="https://your-domain.com"

# Demo Mode (set to 'true' for demo, 'false' for production)
DEMO="false"

üí° Important: Add API Key in TWO Places

You need to add your OpenAI API key in BOTH locations:

  1. Environment File: Add OPENAI_API_KEY="your_key_here" (shown above)
  2. Dashboard Settings: After logging in, go to Settings and add your API key there too

‚ú® Both locations are required for full functionality!

Step 4: Set Up Database

Set up MongoDB database:

Option 1: Local MongoDB

# Install MongoDB locally
# On Ubuntu/Debian:
sudo apt-get install mongodb

# On macOS with Homebrew:
brew install mongodb-community

# Start MongoDB service
sudo systemctl start mongod
# Or on macOS:
brew services start mongodb-community

Option 2: MongoDB Atlas (Cloud)

1. Create a free account at MongoDB Atlas

2. Create a new cluster

3. Get your connection string and add it to your .env.local file

Step 5: Seed Database with Admin User

Create the default admin user in your database:

npm run seed

Default Admin Credentials:
Email: admin@chatai.com
Password: 123456
You can change these credentials after logging in.

Note: The seed script will only create the admin user if it doesn't already exist. If you run it multiple times, it will skip creation and show the existing credentials.

Step 6: Build and Start the Application

Build the application for production:

# Build the application
npm run build

# Start the application
npm start

For development mode:

npm run dev

Troubleshooting

Dashboard Won't Start

Check your environment variables and database connection. Ensure all required dependencies are installed.

Database Connection Error

Verify your MongoDB connection string and ensure MongoDB is running. Check your MONGODB_URI in the .env.local file.

Widget Not Loading

Check that your domain is accessible and the bot.js file is being served correctly.

API Key Issues

Public bots not working: Ensure OPENAI_API_KEY is set in your environment variables.

Dashboard chat errors: Check if users have configured their API keys in Settings, or if the system API key is working.

Invalid API key: Verify your OpenAI API key is valid and has sufficient credits.

Rate limiting: Check your OpenAI usage limits and billing status.
